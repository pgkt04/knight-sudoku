# Knight Sudoku

A Sudoku game with an anti-knight constraint — cells separated by a chess knight's move cannot contain the same digit.

Runs on the web, iOS, and iPadOS.

## How to Play

Standard 9x9 Sudoku rules apply, plus one extra rule: no two cells that are a knight's move apart (an L-shape, like in chess) can contain the same number.

## Development

```bash
npm install
npm run dev
```
